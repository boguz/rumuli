how to use

RUMULI is an Open Source project, so you can use any of it's animation where ever you want. Using the RUMULI animations is really simple. You just need to follow these steps:

Choose

From the animation above chose the one you like

Copy

On the GitBug RUMULI page, copy the animation's HTML and CSS code

Paste

Paste it into your own code and tweek it to your liking.
